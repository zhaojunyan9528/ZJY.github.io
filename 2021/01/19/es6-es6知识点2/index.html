前端

1.变量：–2019/9/26

var:
1.可以重复声明
2.没有块级作用域
3.不能限制

let/const:
1.禁止重复声明
2.限制修改（const)
3.支持块级作用域
    if(true){
        var a= 5;
    }
    alert(a) //正常执行
    if(true){
        let a= 5;
    }
    alert(a) //报错：Uncaught ReferenceError: a is not defined

2.结构赋值：

1.左右两边一样：
    let {a,b} = {a:1,b:2}
    let [a,b] = [a:1,b:2]
2.右边得是个东西：
    let {a,b} = {1,2} // error

3.箭头函数：

() => {}
1.有且仅有一个参数： ()可以省略： param => {}
2.有且仅有一条语句：{}可以省略： i => return i++

修复this

4.收集剩余参数：剩余参数必须是最后一个参数

let json = {a:1,b:2,c:4}
let {a,...b} = json //b:{b:2,c:4}

5.展开参数：

let arr1 = [1,2,3]
let arr2 = [1,2,3]
let arr3 = [...arr1,...arr2]
